## 🚀 Quick Start

### Option 1: Local Setup
1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! No build process needed.

### Option 2: Deploy to GitHub Pages
1. Push this repository to GitHub
2. Go to repository **Settings**
3. Scroll to **GitHub Pages**
4. Select `main` branch as source
5. Your portfolio will be live at: `https://yourusername.github.io/Mujeer_Asghar_Portfolio`

### Option 3: Deploy to Netlify
1. Drag and drop the folder to [Netlify](https://netlify.com)
2. Or connect your GitHub repository
3. Your site will be live instantly

## 🎨 Customization Guide

### Update Personal Information
Edit in `index.html`:
```html
<h1 class="hero-title">Hi, I'm <span class="gradient-text">Your Name</span></h1>
<p>Your email</p>
<p>Your phone</p>
```

### Change Color Scheme
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
    /* ... other colors ... */
}
```

### Add/Edit Projects
Find the projects section in `index.html` and update:
```html
<div class="project-card">
    <div class="project-image"><i class="fas fa-icon"></i></div>
    <div class="project-content">
        <h3>Project Title</h3>
        <p>Project Description</p>
        <!-- ... -->
    </div>
</div>
```

### Update Skills
Modify skill categories in the Skills section:
```html
<div class="skill-category">
    <h3><i class="fas fa-icon"></i> Category Name</h3>
    <div class="skill-tags">
        <span class="skill-tag">Skill 1</span>
        <span class="skill-tag">Skill 2</span>
    </div>
</div>
```
